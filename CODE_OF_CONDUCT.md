# Code of Conduct

This project aims to be welcoming, practical, and respectful for everyone.

## Expected behavior

- Be respectful and constructive.
- Assume good intent and ask clarifying questions before escalating.
- Focus feedback on ideas, code, UX, and evidence.
- Help new contributors get oriented instead of gatekeeping.

## Unacceptable behavior

- Harassment, threats, hate speech, or personal attacks
- Doxxing or sharing private information without consent
- Trolling, sustained disruption, or deliberately bad-faith review behavior
- Sexualized language or unwelcome attention

## Scope

This applies to project spaces, including issues, pull requests, discussions, and review comments.

## Reporting

If you encounter unacceptable behavior, contact the maintainers through GitHub and include links or screenshots when possible. Reports will be reviewed promptly and handled with care.
